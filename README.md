# Physics-Calculation-and-Learning-Support-System
เครื่องมือช่วยคำนวณทางฟิสิกส์สำหรับการเรียนรู้พื้นฐาน รวบรวมการคำนวณสำคัญ เช่น การเคลื่อนที่โปรเจกไทล์ วงจรไฟฟ้า รหัสสีตัวต้านทาน เลนส์ งานและพลังงาน แก๊ส และแรงโน้มถ่วง ผู้ใช้ป้อนตัวแปรแล้วระบบคำนวณแสดงผล พร้อมกราฟและภาพจำลองเพื่อช่วยเข้าใจและใช้งานได้จริง
