เครื่องมือคำนวณทางฟิสิกส์

โปรดป้อนค่าด้วยหน่วย SI: เมตร (m), กิโลกรัม (kg), วินาที (s), แอมแปร์ (A), โวลต์ (V)

หน่วยในผลลัพธ์จะแสดงเป็น ภาษาไทย (ตัวย่ออังกฤษ)

เครื่องกลพื้นฐาน

รวมระบบคาน ระบบรอก ระนาบเอียง ล้อ-เพลา ลิ่ม และสกรู พร้อมสมการและช่องป้อนพารามิเตอร์

การวิเคราะห์สมดุลของคาน

สมดุลโมเมนต์: F₁·L₁ = F₂·L₂ ⇒ F₂ = (L₁/L₂)·F₁

ระบบรอกเดี่ยว

รอกตายตัว: ไม่เพิ่มอัตราทด (เปลี่ยนทิศทางแรง)
รอกเคลื่อนที่ (อุดมคติ): แรงยก F ≈ W/2

ระนาบเอียง

แรงตามระนาบ: F = m·g·sinθ (เมื่อ g = ค่าความเร่งโน้มถ่วง)

ล้อและเพลา

อัตราทดเชิงกล: MA = R / r (R = รัศมีล้อ, r = รัศมีเพลา)

ลิ่ม

อัตราทดโดยประมาณ: MA ≈ L / t (L = ความยาวฐาน, t = ความสูง)

สกรู

อัตราทดโดยประมาณ: MA ≈ (2π·r) / p (p = ระยะต่อเกลียว)

การเคลื่อนที่แบบโพรเจกไทล์

สมการพื้นฐาน: vₓ = v₀ cosθ , vᵧ = v₀ sinθ ; ระยะทาง R = v₀² sin(2θ)/g ; เวลาในอากาศ T = 2 v₀ sinθ / g ; ความสูงสูงสุด hₘₐₓ = vᵧ²/(2g)

พารามิเตอร์เริ่มต้น

กราฟวิถีและแอนิเมชัน

สมการตำแหน่ง: x(t)=v₀ cosθ·t , y(t)=v₀ sinθ·t − ½ g t²

วงจรไฟฟ้า (ไฟฟ้ากระแสตรง)

กฎของโอห์ม: V = I·R ; กำลังไฟฟ้า: P = V·I = I²R = V²/R ; พลังงานไฟฟ้า: E = P·t

กฎของโอห์ม (คำนวณแรงดันและกำลัง)

หาแรงดันไฟฟ้า (V = I·R)

หากระแสไฟฟ้า (I = V / R)

คำนวณค่าไฟฟ้า (Energy & Cost)

ใส่ กำลัง P (W) โดยตรง หรือกรอก V และ I เพื่อคำนวณกำลัง

การถอดรหัสสีตัวต้านทาน (4 แถบ)

แถบ 1–2 = เลขหลัก ; แถบ 3 = ตัวคูณ ; แถบ 4 = ความคลาดเคลื่อน (tolerance)

เลือกสีแถบและคำนวณค่า

ตัวอย่าง

ตัวอย่าง: แดง(2) − ม่วง(7) − ส้ม(×10³) − ทอง(±5%) → R = (27)×10³ = 27 กิโลโอห์ม (kΩ) ±5%

โมเมนตัมเชิงเส้นและการชน

พิจารณา p = m·v , อิมพัลส์ Δp = F·Δt ; การชนแบบยืดหยุ่นและไม่ยืดหยุ่น (1 มิติ)

คำนวณโมเมนตัม

การชนในแนวหนึ่งมิติ (1D)

การเคลื่อนที่แบบวงกลม

แรงสู่ศูนย์กลาง F₍c₎ = m v² / r ; มุมแบงก์ tanθ = v² / (r g)

แรงสู่ศูนย์กลาง

มุมแบงก์

ออปติกส์ — เลนส์บาง

สูตรเลนส์บาง: 1/f = 1/dₒ + 1/dᵢ ; การขยาย M = hᵢ / hₒ = − dᵢ / dₒ ; สูตร Lensmaker สำหรับ f

สมการเลนส์บาง

สูตร Lensmaker (ประมาณ)

แผนภาพลำแสง (Ray diagram) สำหรับเลนส์นูน

ตัวแปร: dₒ = ระยะวัตถุ (m) ; f = ความยาวโฟกัส (m) ; ผลลัพธ์: dᵢ, M และประเภทภาพ (จริง/เสมือน)

งานและพลังงาน

งาน: W = F·s (จูล, J) ; พลังงานจลน์: KE = ½ m v² ; พลังงานศักย์: U = m g h ; กำลัง: P = W / t

คำนวณงาน

พลังงานจลน์

พลังงานศักย์

สมบัติของแก๊สและเทอร์โมไดนามิกส์พื้นฐาน

กฎแก๊สอุดมคติ: pV = nRT (R = 8.314 J·mol⁻¹·K⁻¹) ; พลังงานภายในสำหรับโมโนอะตอม: U = 3/2 nRT ; กฎข้อที่หนึ่ง: ΔU = Q − W

กฎแก๊สอุดมคติ
